Authentication

All endpoints require a valid Bearer token to be included in the Authorization header: 

Authorization: Bearer <your-token>

Endpoints

GET /tasks/all

Retrieve all tasks (both current and completed) for the authenticated user.

Example Request: 
                curl -X GET \
                https://atomapp.up.railway.app/api/tasks/all \
                -H "Authorization: Bearer <your-token>"
Example Response:

GET /tasks/current

Retrieve all current (incomplete) tasks for the authenticated user.

Example Request: 
                curl -X GET \
                https://atomapp.up.railway.app/api/tasks/current \
                -H "Authorization: Bearer <your-token>"
Example Response:
